Detailed Description

A wrapper for the ruby Hash class. This class provides a C++-style interface to ruby's Hash class and its associated rb_hash_* functions. Example:

Hash h;
h[42] = String("foo");
h[10] = String("bar");
std::cout << String(h[42]) << std::endl;
